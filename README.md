# Rotary Assist – Sistema de Gerenciamento de Patrimônio de Equipamentos de Mobilidade

O RotaryAssist é um software desenvolvido para o Rotary Clube de Osvaldo Cruz (SP), uma organização do terceiro setor voltada à assistência social e à saúde comunitária. O sistema tem como objetivo controlar, com rastreabilidade e transparência, todo o ciclo de vida dos equipamentos de mobilidade (como cadeiras de rodas e cadeiras de banho), além de apoiar o atendimento à comunidade.

As principais funcionalidades contempladas no projeto são:
agendamento, empréstimo, devolução, renovação, entrada e baixa de patrimônio, além do registro de manutenções.

## Conteúdo do repositório

**rotary-app** – Interface Front-End da aplicação, desenvolvida em React.

**rotary-backend** – API e banco de dados, desenvolvidos em Node.js, Express e MySQL.
